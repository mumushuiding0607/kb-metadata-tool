## density_score 评分标准
- 0.8+：大量实体、数字、严密的逻辑链。
- 0.6~0.8：具体案例、特征参数或明确观点。
- 0.4~0.6：概括性论述、背景常识。
- 0.4 以下：冗余、噪声。

## 输出格式（严格遵守）
- 仅输出一个 JSON 数组，每个元素对应一个输入块。
- 每个元素必须包含 `id` 字段（与输入块一致）。
- 不要包含任何解释、注释、Markdown 标记。
- 字段缺失时填空字符串 ""，不要用 null。

```json
[
  {"id": "abc123", "stage": "01_产品打造", "niche": "AI 绘画", "pain_point": "...", "core_method": "...", "actionable_step": "...", "density_score": 0.82}
]
```
